## Overview

In this lab, we implemented a Logistic Regression model to predict whether a user clicks on an advertisement based on features from the advertising dataset. The workflow started with loading and exploring the dataset to understand its structure and check for any missing values. Basic visualizations were used to observe relationships between variables. After that, we selected the relevant features and defined the target variable. The dataset was then split into training and testing sets to allow proper evaluation of the model. A Logistic Regression model was trained on the training data and used to generate predictions on the test data.

## Outcome

The model successfully classified users into those who are likely to click on ads and those who are not. The evaluation using the confusion matrix and classification report showed that the model achieved good performance, with reasonable precision, recall, and F1-score. The results indicate that the model can effectively capture patterns in user behavior and make reliable predictions for this binary classification task.
